# issue-triage

Use this skill when `resource-scout` or a manager needs to convert weak signals into a structured Issue Draft.

Required sections:

- Symptom
- Severity
- Reproduction Clues
- Suspected Layer / Owner
- Suggested Labels
- Evidence
