# Technology Stack

## Languages
- **Python**: Core logic for evolutionary loops and hooks.
- **Shell (Bash)**: Scripting and automation for initialization and CLI integration.
- **TypeScript / Node.js**: Required for developing the OpenClaw compatibility layer and plugins.

## Frameworks & Environments
- **Claude Code Plugin API**: The native environment for the original implementation of the framework.
- **OpenClaw Framework**: Target environment for cross-platform compatibility, requiring event-driven hooks and Gateway integration.

## Data Storage
- **Local Files (JSON/Markdown)**: Used for storing system states, user profiles, agent context, and documentation without requiring external databases.
