# Principles

## 格式说明

每条原则使用以下格式：

```markdown
### P-XX: <One-line principle>
- Trigger: <什么情况下触发>
- Constraint (Must/Forbidden): <必须/禁止做什么>
- Verification: <如何验证>
- Exceptions: <什么情况下例外>
- Source: <来源：Issue Log #XX / Date / External Reference>
```
