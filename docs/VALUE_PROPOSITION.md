## How It Works

### Thinking OS (Not Memory System)

Principles isn't a memory plugin — it's a **thinking operating system**. It defines *how* an agent thinks, not just *what* it remembers.

| Component | What It Does | Effect |
|-----------|-------------|--------|
| **Thinking Models** | Injects 10+ mental models from human wisdom | Independent thinking |
| **Principle Internalization** | Turns rules into personality traits | Continuous identity |
| **Evolution Points** | 5-stage growth system (Seed → Forest) | Visible growth trajectory |

### Evolution Points: Growth-Driven, Not Penalty-Driven

The old Trust Engine punished mistakes. Evolution Points rewards growth.

- **Start at 0, only increase** — Your effort is never erased
- **Failure triggers evolution** — Errors teach, they don't penalize
- **5 stages of growth** — Seed → Sprout → Sapling → Tree → Forest
