# SWOT 分析

## 简介

SWOT 分析是一种战略规划工具，用于评估事物的优势、劣势、机会和威胁。

## 核心框架

### Strengths（优势）

内部积极因素：

- 有哪些核心竞争力？
- 拥有哪些独特资源？
- 哪方面做得比竞争对手好？
- 客户为什么选择我们？

### Weaknesses（劣势）

内部消极因素：

- 哪些方面需要改进？
- 缺乏哪些资源？
- 哪些方面不如竞争对手？
- 客户投诉的常见问题是什么？

### Opportunities（机会）

外部积极因素：

- 市场有哪些未满足的需求？
- 行业趋势对业务有何帮助？
- 竞争对手的弱点在哪里？
- 政策环境有何利好？

### Threats（威胁）

外部消极因素：

- 竞争对手有哪些新动作？
- 市场需求是否在变化？
- 政策法规有何限制？
- 技术变革是否带来冲击？

## 分析清单

使用此模型时，依次检查：

- [ ] 是否客观评估了内部优劣势？
- [ ] 是否全面扫描了外部环境？
- [ ] 优势是否足以抓住机会？
- [ ] 劣势是否会加剧威胁？
- [ ] 是否制定了应对策略？

## 常见陷阱

1. **主观偏见**：高估优势、低估劣势
2. **静态分析**：忽视环境动态变化
3. **孤立看待**：未将四要素联系起来
4. **行动缺失**：只分析不行动
