# 第一性原理

## 简介

第一性原理思维（First Principles Thinking）是一种从最基本的事实出发，逐层推导结论的思维方式。通过打破既有假设，重新构建认知框架。

## 核心框架

### 1. 识别核心问题

- 真正要解决的问题是什么？
- 问题的本质是什么？
- 有哪些隐含假设？

### 2. 分解到基本事实

- 什么是不可改变的事实？
- 什么是可以改变的变量？
- 哪些是真正的约束？

### 3. 质疑现有假设

- "我们一直都是这样做的"是否成立？
- 行业惯例是否有更好的替代？
- 哪些约束实际上是自我设限？

### 4. 从头构建解决方案

- 基于基本事实，最优解是什么？
- 如果从零开始设计会怎样？
- 有哪些创新可能？

## 分析方法

**5 Whys 追问法**：
```
问题 → 为什么？ → 为什么？ → 为什么？ → 为什么？ → 为什么？
     ↓           ↓           ↓           ↓           ↓
   表象        原因        深层原因    根本原因    核心真相
```

## 分析清单

使用此模型时，依次检查：

- [ ] 是否识别了问题的本质？
- [ ] 是否分解到了基本事实？
- [ ] 是否质疑了所有假设？
- [ ] 是否从零构建了新方案？
- [ ] 新方案是否可行？

## 常见陷阱

1. **过度分解**：陷入细节，失去方向
2. **忽视约束**：忽略现实可行性
3. **主观判断**：把假设当成事实
4. **分析瘫痪**：过度思考而不行动

## 应用案例

- **SpaceX**：火箭成本高 → 为什么？ → 材料贵 → 材料成本占比仅 2% → 结论：可以大幅降低成本
- **Tesla**：电动车太贵 → 为什么？ → 电池贵 → 为什么？ → 材料贵 → 改用通用材料 → 成本大幅降低
