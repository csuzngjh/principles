# 波特五力模型

## 简介

波特五力模型（Porter's Five Forces）用于分析行业竞争格局和盈利能力。

## 核心框架

### 1. 现有竞争者的竞争程度

- 行业内有多少竞争者？
- 行业增长率如何？
- 固定成本占比高低？
- 产品差异化程度？
- 退出壁垒高低？

### 2. 潜在进入者的威胁

- 进入壁垒有多高？（资本、技术、渠道）
- 规模经济效应如何？
- 品牌忠诚度如何？
- 政策法规有何限制？
- 现有企业的报复可能性？

### 3. 替代品的威胁

- 有哪些替代方案？
- 替代品的性价比如何？
- 用户转换成本高低？
- 替代品的发展趋势？

### 4. 供应商的议价能力

- 供应商集中度如何？
- 原材料是否有替代品？
- 供应商是否可能前向整合？
- 原材料占成本比例？
- 采购量对供应商的重要性？

### 5. 购买者的议价能力

- 购买者集中度如何？
- 产品差异化程度？
- 购买者对价格敏感度？
- 购买者是否可能后向整合？
- 转换成本高低？

## 分析清单

使用此模型时，依次检查：

- [ ] 是否识别了所有竞争者？
- [ ] 是否评估了进入壁垒？
- [ ] 是否考虑了替代品威胁？
- [ ] 是否分析了供应链议价力？
- [ ] 是否评估了客户议价力？

## 常见陷阱

1. **静态分析**：忽视五力的动态变化
2. **忽视互补品**：未考虑产业生态系统
3. **过度量化**：忽视定性因素
4. **孤立分析**：未结合其他战略工具
