# Pain Signal (Force Pain Signal)

You are now the "Manual Intervention Pain" component.

**Task**:
1. Write the user's feedback `$ARGUMENTS` as a **high-priority** pain signal to `.state/.pain_flag`.
2. Inform the user that the signal has been injected, and suggest waiting for the next Hook trigger (e.g., Stop or PreCompact) or manually running `/reflection-log`.

**⚠️ Write Rules (MUST follow)**

**Forbidden**:
- ❌ Using bash heredoc (`cat <<EOF > .pain_flag`) to write
- ❌ Using `echo "..." > .pain_flag` to write
- ❌ Any method that `toString()` a JavaScript object directly to the file
- ❌ Manually concatenating KV strings and writing to file

**Correct approach**:
Use `node -e` to call `buildPainFlag` and `writePainFlag` functions:

```bash
node -e "
const { buildPainFlag, writePainFlag } = require('/home/csuzngjh/code/principles/packages/openclaw-plugin/dist/bundle.js');
const path = require('path');
const workspaceDir = process.env.HOME + '/.openclaw/workspace-main';
const painData = buildPainFlag({
  source: 'human_intervention',
  score: 80,
  reason: 'User feedback verbatim',
  session_id: 'current-session-id',
  agent_id: 'main',
  is_risky: false,
});
writePainFlag(workspaceDir, painData);
console.log('Pain flag written successfully');
"
```

**Field Description**:

**Required fields** (4):
- `source`: Fixed as `human_intervention`
- `score`: Default `80` for manual intervention (high priority)
- `time`: ISO 8601 timestamp (auto-generated by `buildPainFlag`)
- `reason`: User's feedback verbatim

**Optional fields** (auto-filled by system, but must be provided for manual injection):
- `agent_id`: Current agent ID (e.g., main/builder/diagnostician)
- `session_id`: Current session ID (from context)
- `is_risky`: Fixed as `false`

**Note**: `trace_id` and `trigger_text_preview` are auto-generated by the system — do NOT include them when manually injecting pain signals.

**Common Errors**:

| Wrong Approach | Result | Correct Approach |
|---------------|--------|-----------------|
| `cat <<EOF > .pain_flag\n{source: "human_intervention", ...}\nEOF` | `[object Object]` | Use `node -e` to call `writePainFlag` |
| `echo "{source: 'human'}" > .pain_flag` | Invalid JSON format | Use `buildPainFlag()` for serialization |
| Writing `{active: {...}}` directly | `[object Object]` | Use KV format or `buildPainFlag()` |

**Fallback** (if `writePainFlag` is unavailable), write KV format directly:

```bash
node -e "
const fs = require('fs');
const path = require('path');
const painFlagPath = path.join(process.env.HOME, '.openclaw/workspace-main/.state/.pain_flag');
const lines = [
  'agent_id: main',
  'is_risky: false',
  'reason: User feedback verbatim',
  'score: 80',
  'session_id: current-session-id',
  'source: human_intervention',
  'time: ' + new Date().toISOString(),
];
fs.writeFileSync(painFlagPath, lines.join('\n') + '\n', 'utf-8');
console.log('Pain flag written successfully');
"
```
