老板娘是个头发花白的老太太，戴着厚厚的眼镜，正慢悠悠地整理书架。她抬头看了他们一眼："找书？"

"请问，"山姆急切地问，"有没有一个穿紫袍的少年来过？大概十五六岁，会幻术……"

"紫袍？"老板娘推了推眼镜，"哦，那个孩子。他来过，刚走不久。"

莉娜眼睛一亮："他去哪了？"

"不知道，"老板娘耸耸肩，"他来打听几本很偏门的书——什么《记忆封印解法》、《失落的符文大全》，听都没听过。"

"记忆封印？"可可的耳朵竖了起来。

"问得很急，"老板娘回忆着，"我告诉他那些书早就绝版了，他还不死心，非说有人告诉他市集上能找到线索。最后什么都没买，丢下一枚银币就走了。"

"银币？"山姆推了推眼镜。

"问路钱，"老板娘摊摊手，"他说'打扰了'就走了。走得很快，好像在赶时间。"

莉娜若有所思："芬奇在找这些书……不是为了骗钱？"

"他问的那些书都和'记忆'有关。"山姆推了推眼镜，"《记忆封印解法》、《失落的符文大全》……他在找关于记忆封印的资料。"

可可跳到书架上，鼻子抽动："这里有幻术残留，芬奇来过，而且很着急。但他已经走了，这里没有更多线索了。"

"我们先把今天收集到的信息整理一下。"里奥低声说，"芬奇在找记忆封印的书，他腰间有个水晶坠子，他一直望向学院的方向……"

"还有他为什么要用假罗盘骗我。"山姆皱眉，"他怕我抢先找到那些书？"

"这些都和学院有关。"莉娜握紧法杖，"我们先回学院。如果芬奇真的和学院有什么关联，那里会有更多线索。"

可可点了点头："而且伊拉瑞斯老师在钟楼住那么多年，她可能知道些什么。"

> *（三人意识到，那个"骗子"芬奇背后，藏着更深的秘密。而答案，也许就在学院之中。）*

他低头看着手中闪烁的金光，眼神却飘向了学院的方向，目光深邃而复杂。

他的手下意识地摸向腰间，那里挂着一个指甲盖大小的水晶坠子，里面封存着一缕淡金色的光。

*如果传闻是真的……我就能找到答案。*

### 本章学到的概念
| 概念 | 符文名称 | 现实编程 | 故事中的应用 |
|------|----------|----------|--------------|
| 列表 | 空间序列格 | list | 山姆整理的背包内衬，可以随时增删物品，但需按顺序查找 |
| 元组 | 固化属性封印 | tuple | 真正的古董罗盘，其核心属性（名称、年代）是不可改变的 |
| 字符串 | 文字切片与提取 | str | 里奥从宣传单的促销语中，提取出每行的第一个字组成地址 |

### 编程小知识
**列表** 就像一个可以随时增减的魔法背包！
```python
材料清单 = ["月露莓", "星尘草", "龙鳞片"]
材料清单.append("凤凰羽")  # 添加新发现

```

**元组** 就像一个被封印的宝箱，里面的东西不能改变！
```python
罗盘属性 = ("幻术罗盘", "紫色", "指向错误方向")
# 罗盘属性[0] = "真罗盘"  # 错误！元组不能修改
```
