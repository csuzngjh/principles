#### 【伊拉瑞斯的安排】
傍晚，伊拉瑞斯教授从钟楼下来。她的手里紧紧攥着一本泛黄的笔记本。
她没有立刻对三人说话，而是翻开了笔记本的某一页。她的手指在一张古老的星图上停留了一瞬，眼神在夕阳下变得极其柔软，又带着深深的悲伤。
"五十年前……我们也曾站在同样的位置。"她喃喃自语，声音轻得只有风能听见。
然后，她深吸了一口气，合上笔记本，脸上的脆弱瞬间消失，又恢复了那副平静且不容置疑的表情。

"听着，"她神情严肃地对三人说，"我们要做两件事：一是守住校园，二是进禁书区找答案。"

她的语气平静但带着一种不容置疑的力量，就像暴风雨前的宁静。
山姆立刻兴奋起来，"禁书区！太好了！那里有8本关于古代符文的孤本，还有伊拉瑞斯教授亲笔的《风暴纪回忆录》手稿，我早就想……"

他的眼睛闪闪发光，整个人都在颤抖——不是因为恐惧，而是因为即将见到珍贵书籍的纯粹学术狂喜。他从口袋里掏出一个小本子，开始快速记录："禁书区探索清单：1. 带《古符文词典》；2. 带放大镜，. 带防尘口罩；4. 准备足够的羊皮纸做笔记；5..."

可可跳起来用尾巴捂住他的嘴："知道了知道了，你一听到'书'字就两眼放光。现在是紧急情况，不是让你去参观图书馆！"

它叹了口气，对里奥说："你看看他，这就是所谓的'知识就是力量'，但在关键时刻，他的力量只会让他想多借几本书..."

伊拉瑞斯教授微微一笑，但眼中的忧虑并未消散。"禁书区不是普通的图书馆，那里布满了先辈们留下的防护魔法和陷阱。而且……"她顿了顿，"有些知识，不该被轻易触碰。"

里奥注意到教授的目光在自己身上停留了一瞬，那眼神中似乎有某种他读不懂的东西——像是期待，又像是担忧。她似乎想说什么，但最终只是轻轻叹了口气。
*这已经是第二次了，里奥心想，教授看我的眼神总是很奇怪……难道是因为我太慢了，让她担心我会拖累大家吗？*

他不知道的是，伊拉瑞斯教授的目光之所以停留，是因为她在里奥身上看到了某个人的影子——那个曾经为了保护学院而牺牲自己的年轻符文师。
---

#### 【暗处的异动】


就在三人准备离开时，可可的耳朵突然竖了起来。
"等等……"它的声音压得很低，"有人在窥视。"

它的小鼻子在空气中轻轻抽动，眉头皱起，"是风暴残波的味道……但很淡，像是在远处观望。"它望向学院外围的方向，那里有一片被迷雾笼罩的树林，"格罗姆和芬奇……他们的行动太有规律了。每次都在关键节点出现，每次都针对和'记忆'有关的事物。"

"你是说……"山姆推了推眼镜，"他们不只是普通的捣乱者？"

"他们在执行某种计划，"可可的声音变得严肃，"而且这个计划的背后，有一个非常可怕的指挥者。那个指挥者了解学院的每一个弱点，知道在哪里设伏最有效。"

它跳到里奥肩上，轻声说："里奥，你有没有注意到？格罗姆每次战斗时，都会下意识地看一眼自己的双手——那双手在颤抖，不是因为恐惧，而是某种更深的东西。而芬奇……"

可可顿了顿："芬奇每次完成'任务'后，都会望向学院的方向，眼神里有一种我不理解的渴望。就像……就像他也在寻找什么。"

"他们不是真正的敌人，"可可缓缓说道，"他们看起来更像是被困住的人。"

它跳下里奥的肩膀，走到窗边，望着学院深处那座被藤蔓和封印覆盖的高塔。夕阳的余晖中，那座塔的影子被拉得很长，仿佛一个沉睡的巨人。
"里奥，你看到那座塔了吗？"可可的声音突然变得很轻。
里奥顺着它的目光望去，"那是……封印之塔？我听说那里从来没有开放过。"

"它已经有五十年没有开放了，"可可说，目光深邃，"但在五十年前，那里发生过一件事情——一件被所有人遗忘的事情。"

"什么事情？"山姆好奇地问。
可可没有回答，只是轻声说："有些记忆，被封存是有原因的。但当风暴残波开始聚集，当有人试图重启那场灾难……那些被封存的记忆，迟早会苏醒。"

它转过身，认真地看着里奥："里奥，当那一天到来的时候，你需要做出选择。一个关乎学院、关乎你家族、甚至关乎……"它顿了顿，"关乎我的选择。"

"关乎你？"里奥困惑地看着可可。
可可只是摇了摇头，没有再说什么。但里奥注意到，它的目光再次飘向了那座封印之塔，眼中闪过一丝他读不懂的情绪——像是期待，又像是恐惧。
*可可...你到底在隐瞒什么？*

那个身影穿着漆黑的长袍，脸上戴着一副银色的面具，看不清容貌。他的手中握着一颗黑色的水晶球，里面翻涌着紫色的雾气。

"那个孩子……"他的声音沙哑而低沉，"他的符文编织方式……和他一模一样。"

水晶球中的雾气翻涌得更剧烈了，仿佛在回应他的情绪。
"五十年了，"黑袍人低声说，"我终于找到了那个血脉。这一次，不会再有人阻止我了。"

他的目光穿过学院的建筑，落在那座封印之塔上。
"沉睡者们……准备醒来吧。"

水晶球中，传来一阵若有若无的笑声——那不是一个人的笑声，而是无数声音交织在一起的回响，令人毛骨悚然。

### 编程小知识
**函数** 就像一个可以重复使用的魔法工具！
```python
def 检测风暴残波(学生):
    嗅觉检测(学生)
    能量扫描(学生)
    记忆测试(学生)
    return 检测结果

# 只需要调用函数名，就能重复使用
结果1 = 检测风暴残波(艾伦)
结果2 = 检测风暴残波(大厨)
```

**模块** 就像一个整理好的工具箱！
```python
from 分析工具 import 检测风暴残波, 能量扫描
from 记录工具 import 保存数据, 生成报告

# 直接使用工具箱里的工具
检测风暴残波(学生)
```
