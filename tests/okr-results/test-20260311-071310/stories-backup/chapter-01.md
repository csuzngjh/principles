#### 【里奥的"静音"符文】

- 如果格罗姆的锤子砸向岩壁，需要最强的静音效果
- 如果魔蝎发出嘶鸣，需要中等强度的静音
- 如果只是普通的脚步声，轻微的静音就够了

"这就是分岔符文的原理，"可可在里奥耳边轻声说，"如果遇到这种情况，就这样做；如果遇到那种情况，就那样做。"

里奥突然明白了什么："所以……我在做的不仅是魔法，也是一种思考方式？"

"聪明，"可可的尾巴得意地摆动，"这就是为什么我要教你符文逻辑思维。魔法也好、符文逻辑也好，本质都是学会清晰地思考。"

里奥缓缓编织出静音符文，像把一张透明的布铺开。符文根据声音的类型和强度自动调整效果。莉娜在无声的空间里释放火焰，火舌像极光般贴着岩壁滑过，既明亮又克制。

## 📖 符文笔记

### 本章学到的概念
| 概念 | 符文名称 | 现实编程 | 故事中的应用 |
|------|----------|----------|--------------|
| 条件判断 | 分岔符文 | if/elif/else | 里奥根据声音类型调整静音符文强度 |
| 循环 | 周而复始符 | for/while | 可可的痒痒粉让格罗姆持续抓痒 |

### 动手思考
1. 里奥用分岔符文"处理了三种不同的情况。你能想到生活中还有哪些"如果...就.."的情境吗？2. 山姆说12小于20，检查通过"，这用了什么比较符？
### 编程小知识
在编程中，`if/else` 就像魔法中的"分岔符文"，让程序可以根据不同情况做出不同反应。例如：

```python
if 声音类型 == "锤子":
    静音强度 = "最开
elif 声音类型 == "魔蝎":
    静音强度 = "中等"
else:
    静音强度 = "轻微"
```
