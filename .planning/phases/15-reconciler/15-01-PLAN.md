  <files>packages/openclaw-plugin/src/core/pd-task-reconciler.ts</files>
  <instructions>
Create `packages/openclaw-plugin/src/core/pd-task-reconciler.ts` with:

1. **Types:**
   - `ReconcileResult { created: string[]; updated: string[]; skipped: string[]; errors: ReconcileError[] }`
   - `ReconcileError { taskId: string; message: string }`
   - `ReconcileOptions { dryRun?: boolean; workspaceDir: string; logger?: any }`
   - `CronStoreFile { version: number; jobs: CronJob[] }` (read-only, from cron/jobs.json)
   - `DiffAction { type: 'CREATE' | 'UPDATE' | 'DISABLE' | 'ORPHAN' | 'SKIP'; task?: PDTaskSpec; job?: CronJob }`

2. **readCronStore(workspaceDir: string): Promise<CronStoreFile>:**
   - Reads `~/.openclaw/cron/jobs.json`
   - Returns `{ version: 1; jobs: [] }` if file doesn't exist
   - Throws on parse error

3. **writeCronStore(workspaceDir: string, store: CronStoreFile): Promise<void>:**
   - Uses `withLockAsync` to protect write
   - Atomic: write to temp file, then rename
   - Path: `~/.openclaw/cron/jobs.json`

4. **diff(declared: PDTaskSpec[], actual: CronJob[]): DiffAction[]:**
   - For each declared task (by name prefix "PD "):
     - NOT in actual → CREATE
     - IN actual, enabled, version mismatch → UPDATE
     - IN actual, enabled, version match → SKIP
     - IN actual, NOT enabled → DISABLE
   - For each actual job with "PD " prefix:
     - NOT in declared → ORPHAN (log warning)
   - Return array of DiffAction

5. **buildCronJob(task: PDTaskSpec, nowMs: number): CronJob:**
   - Maps PDTaskSpec → CronJob structure
   - Sets name = task.name (must start with "PD ")
   - Sets sessionTarget = 'isolated', wakeMode = 'now'
   - Sets payload.kind = 'agentTurn', lightContext = task.execution.lightContext
   - Sets delivery.mode = task.delivery.mode
   - Sets createdAtMs/updatedAtMs = nowMs
   - Sets metadata.pdVersion = task.version, metadata.pdTaskId = task.id
   - Sets state = { nextRunAtMs: nowMs + task.schedule.everyMs }

6. **reconcilePDTasks(workspaceDir: string, options?: Partial<ReconcileOptions>): Promise<ReconcileResult>:**
   - Reads pd_tasks.json via PDTaskStore
   - Merges BUILTIN_PD_TASKS with pd_tasks.json (pd_tasks.json overrides builtin by id)
   - Reads cron/jobs.json
   - Computes diff
   - If NOT dryRun: applies CREATE/UPDATE/DISABLE with atomic write
   - Updates pd_tasks.json meta (lastSyncedAtMs, lastSyncStatus, lastSyncedJobId)
   - Returns ReconcileResult

7. **Dry-run mode:** When `options.dryRun === true`, skip all writes, return diff result only

Follow the file structure and patterns from cron-initializer.ts for reading/writing jobs.json.
Use withLockAsync from file-lock.ts for all writes.
</instructions>
