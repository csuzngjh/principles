# Plan 32-02: Evidence-Trace Hardening for Replay and Promotion

## Goal

Ensure replay and promotion outputs distinguish observed evidence from empty or proxy evidence.

## Scope

- Make empty replay sample sets resolve to `needs-review` instead of `pass`
- Add evidence summary to replay reports
- Add evidence mode summary to promotion gate results
- Surface evidence mode in rollout evaluation output
- Add regression tests for empty replay evidence and eval-proxy promotion evidence

## Success Criteria

1. Replay reports never present empty evidence as a passing result
2. Promotion gate results expose whether constraints came from shadow evidence or eval proxy
3. Operator-visible output includes evidence mode at decision time
4. Tests lock the new semantics in place
