# Plan 32-01: Harden ORPO Export Against Fabricated Facts

## Goal

Ensure ORPO export emits only evidence-bounded narratives when source lineage is missing or sparse.

## Scope

- Add structured evidence summary to exported ORPO samples
- Derive neutral fallback prompt/rejected/rationale text when lineage evidence is missing
- Preserve export success for review-approved samples while removing unsupported claims
- Add tests for unknown-evidence and observed-evidence export cases

## Out of Scope

- Replay engine narrative hardening
- Promotion report/operator UI wording
- Artifact-generation prompt changes

## Success Criteria

1. Exported JSONL includes machine-readable evidence summary per sample
2. Missing lineage is represented as `unknown`, not inferred as success or failure
3. Export text does not narrate pain/failure/violation details when evidence is absent
4. Tests cover both unknown-evidence and observed-evidence paths
