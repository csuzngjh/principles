<action>

**New Methods:**

1. **getCommitByTaskId(taskId: string): Promise\<CommitRecord | null\>**
   - Query `commits` table WHERE task_id = ? ORDER BY created_at DESC LIMIT 1
   - Returns CommitRecord with commitId, artifactId, taskId, runId, status
   - Needed by diagnose status to return commitId/artifactId for succeeded tasks

2. **getCandidatesByTaskId(taskId: string): Promise\<CandidateRecord[]\>**
   - JOIN: tasks → runs → commits → principle_candidates WHERE tasks.task_id = ?
   - Per D-05: candidates are reached via tasks→runs→commits→principle_candidates chain
   - Return array of candidate records (candidateId, artifactId, taskId, sourceRunId, title, description, confidence, status, createdAt)

3. **getCandidate(candidateId: string): Promise\<CandidateRecord | null\>**
   - Query principle_candidates WHERE candidate_id = ?
   - Returns single candidate or null

4. **getArtifact(artifactId: string): Promise\<ArtifactRecord | null\>**
   - Query artifacts WHERE artifact_id = ?
   - Returns artifact record (artifactId, runId, taskId, artifactKind, contentJson, createdAt)

5. **getArtifactWithCandidates(artifactId: string): Promise\<ArtifactWithCandidates | null\>**
   - Query artifacts WHERE artifact_id = ? + JOIN principle_candidates
   - Per D-06: single query with JOIN, returns artifact + inline candidates array
   - Define ArtifactWithCandidates interface: `{ artifact: ArtifactRecord; candidates: CandidateRecord[] }`

**Types to define (inline in RuntimeStateManager or in a new types file):**

```typescript
interface CommitRecord {
  commitId: string;
  taskId: string;
  runId: string;
  artifactId: string;
  idempotencyKey: string;
  status: string;
  createdAt: string;
}

interface CandidateRecord {
  candidateId: string;
  artifactId: string;
  taskId: string;
  sourceRunId: string;
  title: string;
  description: string;
  confidence: number | null;
  sourceRecommendationJson: string;
  status: 'pending' | 'consumed' | 'expired';
  createdAt: string;
}

interface ArtifactRecord {
  artifactId: string;
  runId: string;
  taskId: string;
  artifactKind: string;
  contentJson: string;
  createdAt: string;
}

interface ArtifactWithCandidates {
  artifact: ArtifactRecord;
  candidates: CandidateRecord[];
}
```

**SQL patterns (use db.prepare().get() for single row, db.prepare().all() for multiple):**
```typescript
// getCandidatesByTaskId - needs JOIN through runs and commits
const candidates = db.prepare(`
  SELECT pc.candidate_id, pc.artifact_id, pc.task_id, pc.source_run_id,
         pc.title, pc.description, pc.confidence, pc.status, pc.created_at,
         pc.source_recommendation_json
  FROM principle_candidates pc
  JOIN commits c ON c.artifact_id = pc.artifact_id
  JOIN runs r ON r.run_id = c.run_id
  JOIN tasks t ON t.task_id = r.task_id
  WHERE t.task_id = ?
  ORDER BY pc.created_at DESC
`).all(taskId) as CandidateRow[];
```

**Important:** All methods must call `this.assertInitialized()` at the start. Use `this.connection.getDb()` for direct SQL queries since candidates/artifacts tables are not managed by TaskStore/RunStore.
