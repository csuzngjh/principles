## Locked Decisions from Phase 39 Context

### FPR Counter Schema (D-39-01, D-39-02)
- Each `CorrectionKeyword` gains three new counters: `hitCount`, `truePositiveCount`, `falsePositiveCount`
- Counters initialized to 0 on seed keywords, incremented atomically on each match/feedback

### Confidence Score Formula (D-39-03, D-39-04)
- `match().score = keyword.weight x (TP / (TP + FP + 1))`
- +1 smoothing in denominator avoids division by zero

### Weight Decay Function (D-39-15, D-39-16, D-39-17)
- `keyword.weight = keyword.weight x 0.8` on each confirmed FP
- Decay applied immediately in `recordFalsePositive()` before flush to disk
- Keywords at very low weight (e.g., < 0.1) can still match but contribute minimally

### Throttle Scope (D-39-12, D-39-13)
- Per-workspace throttle: each workspace has independent 4 calls/day quota
- Use `checkCooldown(wctx.stateDir, ..., maxRunsPerWindow: 4, quotaWindowMs: 24 * 60 * 60 * 1000)`
</context>

### recordFalsePositive(term: string)
```typescript
recordFalsePositive(term: string): void {
  const keyword = this.store.keywords.find(k => k.term.toLowerCase() === term.toLowerCase());
  if (!keyword) return;

  keyword.falsePositiveCount = (keyword.falsePositiveCount ?? 0) + 1;

  // D-39-15: Multiplicative weight decay x0.8 on confirmed FP
  keyword.weight = keyword.weight * 0.8;
  // D-39-17: Minimum weight floor — allow very low weights to persist
  // (keywords at weight < 0.1 still match but contribute minimally)

  // Update in-store reference
  const keywordIndex = this.store.keywords.findIndex(k => k.term.toLowerCase() === term.toLowerCase());
  if (keywordIndex >= 0) {
    this.store.keywords[keywordIndex] = { ...keyword };
  }

  // D-39-16: Apply decay BEFORE flush to disk
  this.flush();
}
```

### recordTruePositive(term: string)
```typescript
recordTruePositive(term: string): void {
  const keyword = this.store.keywords.find(k => k.term.toLowerCase() === term.toLowerCase());
  if (!keyword) return;

  keyword.truePositiveCount = (keyword.truePositiveCount ?? 0) + 1;

  // Update in-store reference
  const keywordIndex = this.store.keywords.findIndex(k => k.term.toLowerCase() === term.toLowerCase());
  if (keywordIndex >= 0) {
    this.store.keywords[keywordIndex] = { ...keyword };
  }

  this.flush();
}
```

Both methods should be case-insensitive on term matching.
