   ```yaml
           eventType: nocturnal_artifact_persisted

           eventCategory: evaluated
           statsField: evolution.ruleHostEvaluated
   ```

   - `summary.workflowFunnels[0].funnelKey === 'nocturnal'`
   - `summary.workflowFunnels[0].stages[0].label === 'dreamer_completed'`

   - `summary.workflowFunnels[1].stages[0].label === 'evaluated'`
   - `summary.workflowFunnels[1].stages[0].count === 15`
8. Cleans up with `loader.dispose()`

This test exercises the complete path: YAML file -> WorkflowFunnelLoader.load() -> getAllFunnels() -> RuntimeSummaryService.getSummary() -> workflowFunnels output.
