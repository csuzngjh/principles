### Task 2: Create `ResilientContextAssembler`

**Files:**
- CREATE `src/runtime-v2/store/resilient-context-assembler.ts`
- MODIFY `src/runtime-v2/index.ts` — add exports

**Details:**
```typescript
export class ResilientContextAssembler implements ContextAssembler {
  constructor(
    private readonly inner: ContextAssembler,
    private readonly emitter: StoreEventEmitter,
  ) {}

  async assemble(taskId: string): Promise<DiagnosticianContextPayload> {
    try {
      return await this.inner.assemble(taskId);
    } catch (error) {
      return this.buildDegradedPayload(taskId, error);
    }
  }
}
```

- `buildDegradedPayload()` catches any error, returns valid DiagnosticianContextPayload:
  - Synthetic UUID contextId
  - SHA-256 of `"degraded"` as contextHash (deterministic marker)
  - `taskId` from input
  - `workspaceDir: ''` (unknown)
  - `sourceRefs: [taskId]`
  - `diagnosisTarget: {}` (empty)
  - `conversationWindow: []`
  - `ambiguityNotes: [error description + degradation message]`
- Emits `degradation_triggered` telemetry event with payload:
  - `component: 'ContextAssembler'`
  - `trigger: error category or message`
  - `fallback: 'degraded_payload'`
  - `severity: 'error'` for schema failures, `'warning'` for data-not-found

### Task 3: Create `ResilientHistoryQuery`

**Files:**
- CREATE `src/runtime-v2/store/resilient-history-query.ts`
- MODIFY `src/runtime-v2/index.ts` — add exports

**Details:**
```typescript
export class ResilientHistoryQuery implements HistoryQuery {
  constructor(
    private readonly inner: HistoryQuery,
    private readonly emitter: StoreEventEmitter,
  ) {}

  async query(trajectoryRef: string, cursor?: string, options?: HistoryQueryOptions): Promise<HistoryQueryResult> {
    if (cursor) {
      try {
        return await this.inner.query(trajectoryRef, cursor, options);
      } catch (error) {
        // Cursor invalid — fall back to first page
        this.emitDegradation('HistoryQuery', 'invalid_cursor', 'first_page_fallback');
        return this.inner.query(trajectoryRef, undefined, options);
      }
    }
    return this.inner.query(trajectoryRef, undefined, options);
  }
}
```

- On cursor error: catch, emit telemetry, re-query without cursor (first page)
- Non-cursor queries pass through unchanged (already graceful)
