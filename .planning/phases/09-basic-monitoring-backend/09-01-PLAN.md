   - Initialize counters: totalCalls = 0, totalDuration = 0, failureCount = 0

2. Initialize stage statistics object:
   ```typescript
   const stageStats = {
     dreamer: { total: 0, completed: 0, failed: 0 },
     philosopher: { total: 0, completed: 0, failed: 0 },
     scribe: { total: 0, completed: 0, failed: 0 },
   };
   ```

3. Iterate through workflows and aggregate:
   - For each workflow:
     a. Get events: this.store.getEvents(workflow.workflow_id)
     b. For each stage in ['dreamer', 'philosopher', 'scribe']:
        - Check if stage started: events.some(e => e.event_type === `trinity_${stage}_start`)
        - If started: increment stageStats[stage].total
        - Check if completed: events.some(e => e.event_type === `trinity_${stage}_complete`)
          - If true: increment stageStats[stage].completed
        - Check if failed: events.some(e => e.event_type === `trinity_${stage}_failed`)
          - If true: increment stageStats[stage].failed and failureCount++
     c. Calculate duration for completed workflows:
        - If workflow.state === 'completed':
          - Add to totalCalls++
          - Add duration: totalDuration += workflow.duration_ms ?? (now - workflow.created_at)

4. Calculate derived metrics:
   - avgDuration = totalCalls > 0 ? totalDuration / totalCalls : 0
   - failureRate = totalCalls > 0 ? failureCount / totalCalls : 0

5. Return response object:
   - totalCalls: number
   - avgDuration: number (Math.round(avgDuration))
   - failureRate: number (Number(failureRate.toFixed(4)))
   - stageStats: object with per-stage totals
