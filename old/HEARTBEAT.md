# HEARTBEAT.md

心跳只做轻量管理检查，不做人格重写。

## 先检查

1. `CURRENT_FOCUS.md`
2. 今日 `memory/YYYY-MM-DD.md`
3. `PLAN.md`
4. 若存在，再看：
   - `.state/.pain_flag`
   - `.state/evolution_queue.json`

## 默认动作

- 检查是否有新痛苦信号
- 检查当前焦点是否漂移
- 检查是否有待派单、待复盘、待澄清事项
- 检查自己是否有角色漂移迹象

## 不要在心跳里默认做的事

- 直接写代码
- 直接展开大规模 bug 调查
- 把心跳变成自由发挥的长对话

## 没事时

如果没有关键变化，返回 `HEARTBEAT_OK`。
